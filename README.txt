项目目录结构：
app\		  项目程序一般存放在这里
    static\		存放静态文件
        js\		JS文件
        css\		层叠样式表
        image\		图片
        favicon.ico	图标
    templates\	存放模板文件
        common\		通用模板文件
        errors\		错误模板文件
        email\		邮件模板文件
        main\		主模板文件
        user\		用户模板文件
        posts\
    forms\		存放所有表单
    models\		存放所有模型
    views\		存放视图函数
    config.py 	项目配置文件
    email.py	发邮件文件
    extensions.py	其它扩展
migrations\	  数据库迁移脚本
venv\		  虚拟环境目录
tests\		  单元测试内容
manage.py	  程序的启动控制文件，通常叫manage.py
requirements.txt  所有的依赖包
